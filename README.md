# React + Spring Boot Backend

This repository contains the **backend API** of a full-stack application built with **Spring Boot 3**, **Java 17**, and an **H2 in-memory database**.  
It provides secure, RESTful CRUD endpoints for **Products**, ready to be consumed by a React frontend.
